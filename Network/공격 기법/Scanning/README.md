# **스캐닝(Scanning)이란?**

### `Scanning`이란 네트워크를 통해 제공하고 있는 서비스, 포트, Host 정보 등을 알아내는 것을 의미하며, 해당 대상의 취약점을 찾기 위한 해킹의 **초기 단계**이다.

<br>

###  이 때 공격자는 대상 시스템을 최대한 흔적을 남기지 않고 스캔 하는 것을 목적으로 하는데, 흔적을 남기지 않기 위해 다양한 방식의 Scan 방법이 개발 되었다.
