# **스니핑(Snifing)이란?**

### `Sniffing`이란 사전적 의미로는 '코를 킁킁거리다', '냄새를 맡다' 등의 뜻으로, 네트워크 상에서 자신이 아닌 다른 상대방의 패킷(데이터) 교환을 엿듣는 것, 즉 **도청하는 것을 의미**한다.

<br>

###  네트워크 트래픽을 `도청(eavesdropping)`하는 과정을 의미하며, 스니핑을 할 수 있는 도구를 `스니퍼(sniffer)`라고 한다.
